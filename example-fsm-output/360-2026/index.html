State Machines

Extracted from RainMaker26 robot code

  • SuperStructure
    Top-level orchestrator FSM that coordinates all subsystems. Maps driver-requested modes to internal states controlling shooter, intake, indexer, and hopper.
    Files: SuperStructure.java
  • Shooter State Machine
    Coordinates flywheel, hood, and kicker to determine shooter readiness. Event-driven firing cycle with disturbance tolerance.
    Files: ShooterStateMachine.java
  • Intake State Machine
    Coordinates intake rollers and pivot for intaking, stowing, agitating, and reversing. Driven by driver inputs through SuperStructure.
    Files: IntakeStateMachine.java
  • Target Selection
    Decides whether to shoot at the hub or outpost based on manual overrides or automatic alliance zone detection.
    Files: TargetSelectionStateMachine.java
  • Flywheel
    Bang-bang velocity controller with shot detection, recovery tracking, and underspeed detection for rapid-fire scenarios.
    Files: Flywheel.java
  • Flywheel Kicker
    Bang-bang velocity controller for the kicker wheel, mirroring the flywheel's shot detection and recovery lifecycle.
    Files: FlywheelKicker.java
  • Hood
    Position controller for the hood angle, with aiming, ducking, and soft-limit-aware PID switching.
    Files: Hood.java
  • Intake Pivot
    Position controller with stow/deploy, oscillating agitation cycles, and progressive step-down agitation.
    Files: IntakePivot.java
  • Intake Roller
    Roller motor controller with automatic jam detection and multi-attempt unjamming recovery.
    Files: IntakeRoller.java
  • Hopper Sensor
    Sensor state with hysteresis latch for agitation mode. Detects ball presence over the spindexer.
    Files: HopperSensor.java
  • Indexer
    Simple wanted-state-driven motor controller for indexing balls into the shooter.
    Files: Indexer.java
  • Hopper Roller
    Simple wanted-state-driven motor controller for the hopper agitation roller.
    Files: HopperRoller.java
